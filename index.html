carousel

the carrousel is a system a little bit complex than others.
you will need to create a div with the class "carousel"
then inside of this div, you will need to create a "main" element, and this element need to have id "carousel" and class "carousel-load"
then inside of "main" you will need to create 5 div with the class "item".
↑ This steps above will create the items

↓ this steps below will create the radio buttons
for create the radio buttons, inside of the div with the class "carousel",
you will need to create 5 input elements with the type "radio" and name "position"
and then, one of them need to be with the "checked" attribute.
